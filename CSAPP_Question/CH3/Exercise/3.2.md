For each of the following lines of assembly language, determine the appropriate instruction suffix based on the operands. (For example, mov can be rewritten as movb, movw, movl, or movq.)  

```
mov? %eax, (%rsp)            // ? = l
mov? (%rax), %dx             // ? = w
mov? $0xFF, %bl              // ? = l(x) b(o)
mov? (%rsp,%rdx,4), %dl      // ? = b
mov? (%rdx), %rax            // ? = q
mov? %dx, (%rax)             // ? = w
```
